Login Widget: no frame integration guide

Example Script

Below is a minimal example html structure showing the options you need to set to load the application into a container <div>. Please customise the ID of the widget container and ID of the application container to suit your needs. Also, add your own css and locale as additional parameters if needed..

<!doctype html>
<html lang="en">
 <head>
  <meta charset="utf-8" />
 </head>
 <body>
  <div id="lnut-login-widget">
   <p><a href="http://www.languagenut.com/en/login/">Log In to LanguageNut</a></p>
   <p><a href="http://www.languagenut.com/en/trynow/">Try LanguageNut Now</a></p>
   <p><a href="http://www.languagenut.com/en/subscribe/school/">Free trial</a></p>
  </div>
  <div id="lnut-application"></div>
  <script src="http://code.jquery.com/jquery-1.5.1.min.js"></script>
  <script src="http://www.languagenut.com/scripts/lnut-login-widget.js"></script>
  <script type="text/javascript">
   if(window.jQuery) {
    $(document).ready(function(){
     $('#lnut-login-widget').lnutLoginWidget({
      option: 'noframe',
      value:  'lnut-application'
     });
    });
   }
  </script>
 </body>
</html>
