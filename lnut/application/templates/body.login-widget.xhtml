Login Widget

Overview

Our login widget will allow you to access the LanguageNut platform from a form included on your site.

This widget can allow you to embed the services languagenut offers within the theme of your site without having your users, members or customers leave your site.

Generally, this widget would be included on a page within your content management system, school management system or other platform after your users have logged in.

General Guidelines

The widget is not designed to comply with accessibility guidelines for web-based content, but a modified version of the code can be created and customised for your specific needs. Please contact us if you have concerns about accessibility.

We use JavaScript to embed the widget code onto the webpage. If JavaScript is disabled in the environment your users are using or their web browsers do not support JavaScript, then the features of this widget will not be available to them. Please contact us for alternative options if this is the case. Also bear in mind, JavaScript is used to embed the flash application into the web pages as well, so the flash application will also be unavailable without JavaScript support.

Get this widget for your site

We have written this widget as a jQuery plugin and recommend that websites that use jQuery upgrade to the latest version to ensure compatibility with the plugin we have written. At the time of writing this is version 1.5.

How to use this widget

The login widget is designed to be added to your site with minimal coding skills. By including one piece of JavaScript code to your site and configuring which behaviour you would like to employ after a member has logged in, you can instruct the widget to include the languagenut application in an iframe, within an area on your page, as a popup or to submit the login information directly to languagenut. Please pick the option that most closely matches your needs and let us know if you need guidance.

Styling your Widget

When the widget is added to your site, it adds a stylesheet, hosted on our servers, to your page. All of the declarations in the stylesheet can be replaced by overriding the default stylesheet option. Please see below for an example.

Plugin Options

Parameter Default Value Description
locale en The language in which the application should be presented.
option direct Valid options are 'iframe', 'noiframe', 'popup' or 'direct'.
value When using an iframe or container, this must be the value of the id attribute on the element where the content should be loaded.
css lnut-login-widget.css If you leave this blank, the default style will be used, as shown above, but you can also use your own stylesheet. Please use the full url to the stylesheet.
usecss yes If you disable the loading of the stylsheet, then no additional stylesheet will be loaded. Use this option if you want to customise the styles within your own stylsheet.
